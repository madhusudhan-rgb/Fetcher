NET - API & Web Fetcher

NET is a lightweight API and web data fetching tool built with **Python + KivyMD**.

It allows you to enter any URL and inspect the response, making it useful for learning HTTP requests, testing APIs, and exploring web data.

Features

- Fetch data from any URL
- API response testing
- JSON response formatting
- HTML page inspection
- Extract webpage links
- Response time tracking
- HTTP status code display
- Error handling
- Dark Material UI
- Scrollable response viewer

Dont forget to  pip install all the modules
